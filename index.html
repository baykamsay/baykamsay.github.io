Projects

Droneer

Game to teach students programming by writing code for drones to complete objectives through space. It aims to provide a more fun and engaging alternative to conventional ways of learning programming. The game has two core parts. It has an IDE part where the players can write the code for their drone in Java and a in game part where the player's drone executes the code. I was the lead developer in a team of 6 and wrote the core functionalities of the drones, along with most of the in game part. Written in Java, HTML, and CSS. 

SpaceX Launches

Single page application to monitor the last and the next rocket launch from SpaceX. It uses the SpaceX API: https://github.com/r-spacex/SpaceX-API and shows the countdown to the next rocket launch and the time of the last rocket launch depending on the choice of the user. Written in HTML, CSS, and JavaScript.

Caculator

A simple calculator written in JavaScript that can do addition, subtraction, multiplication, and division. It can also clear the screen and delete one digit.
